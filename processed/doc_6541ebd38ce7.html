The minimum standards, provisions and requirements for safe and stable design, methods of construction and usage of materials in buildings and structures enlarged, altered, repaired, moved, converted to other uses or demolished, and the equipment, maintenance, use and occupancy of all buildings in the unincorporated parts of the county to which the state building code applies shall be regulated in accordance with the terms of the latest edition of the state building code, which is hereby adopted and incorporated in this chapter by reference as though fully set forth herein, and a certified copy of such code shall be on file in the county offices.
