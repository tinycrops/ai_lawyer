(a)

It shall be the duty of the department of permits and inspections to enforce all of the provisions of this chapter and of the regulatory codes adopted in this chapter, and to make all inspections necessary to determine whether or not the provisions of this chapter and such codes are being met.

(b)

The state building code (general construction), the state uniform residential building code and the state regulations for mobile homes shall be enforced by the building inspector. The state plumbing code shall be enforced by the plumbing inspector. The state heating code shall be enforced by the heating-air conditioning inspector. The state electrical code shall be enforced by the electrical inspector.
